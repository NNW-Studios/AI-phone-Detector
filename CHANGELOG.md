# Changelog

## 4.1.0 - Apple Silicon Release

- Added a macOS ARM64-only app variant.
- Added Apple Metal/MPS inference path for YOLOv8.
- Added Streamlink-based Twitch extraction with yt-dlp fallback.
- Added Core ML export helper for later Neural Engine experiments.
- Added macOS installer and generated launcher.
- Removed Windows/CUDA assumptions from the macOS code path.
